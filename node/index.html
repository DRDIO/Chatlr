Commands
Chatlr supports the following commands:

  • /help
  • /topic
  • /away
  • /ignore USERNAME
  • /mobile
  • /night

Rules
Failure to abide by the following rules will result in escalating ban:

  • No Bullying
  • No Griefing
  • No Whoring
  • No Vulgarity
  • No Spamming

Problems
Chatlr works in Explorer, Firefox, Safari and Chrome. Please report all problems to the Ask Box along with your OS, browser version, and any error messages.

About
Chatlr was created by Kevin Nuut using Tumblr OAuth, Socket.IO, Node.JS, and jQuery. We'd love it if you followed the Blog or Recommend Chatlr as a startup!
